Spotify User Churn Analysis using SQL

Project Overview

This project analyzes Spotify user behavior and customer churn using SQL.

Objectives

- Analyze user engagement patterns
- Calculate churn rates
- Compare active and churned users
- Identify factors influencing churn
- Generate business insights

SQL Concepts Used

- Aggregate Functions
- GROUP BY
- CASE WHEN
- Subqueries
- CTEs
- Views
- Window Functions (ROW_NUMBER, RANK, DENSE_RANK)

Key Insights

- Free users showed the highest churn rate.
- Student users had the lowest churn rate.
- Mobile users had better retention than Web users.
- Higher skip rates were associated with churned users.
- Active users had higher listening time than churned users.

Tools Used

- MySQL Workbench
- SQL
